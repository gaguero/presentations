# Dolphin Blue Paradise - Interactive Sitemap

An interactive, strategic sitemap visualization for the Dolphin Blue Paradise eco-resort website.

## Features

- 7-page optimized site structure
- Interactive hover tooltips
- Detailed modal views with:
  - Page layout and internal structure
  - Strategic rationale and conversion optimization
  - SEO strategy and user psychology insights
- Per-page comment system for feedback
- Responsive design

## View the Sitemap

Open `sitemap-diagram.html` in your browser or visit the deployed version.

## Structure

- **7 Main Pages**: Home, About, Accommodations, Dining, Experiences, Sustainability, Blog, Contact
- Each page includes detailed content hierarchy and strategic reasoning
- Integrated sections (Packages, FAQs, Getting Here) are merged into parent pages for optimal UX

## Technology

- Pure HTML/CSS/JavaScript
- SVG-based diagram rendering
- LocalStorage for comments persistence
- Mobile-responsive layout
